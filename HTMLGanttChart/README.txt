*LCC Gantt Chart Utility*

- Copy the py, csv and png files to a folder on the Programming Env.
- Generate html gantt charts by adding your activities to the gantt.csv 
- Open gcg.py in Notepad++ and select "Plugins > PyNPP > Run File in Python"
- Edit the html to change your candidate number and the project plan version & date.
- Open in Chrome & "Save as PDF" option to then save to pdf format.
- Do 3 of these charts, showing progression / changed dates and added activities:
-- 1 from early on in the project (e.g. Oct, main stages & activities)
-- 1 from approx Dec / Jan (more detail)
-- 1 from start of March with more detail
